Это WPF-приложение представляет собой систему управления личными финансами с расширенными функциями безопасности и аналитики.

Ключевые возможности:

Безопасность: Ролевой доступ (Админ/Пользователь), надежное хеширование паролей, защита от брутфорса и смена пароля.

Управление данными:

Для администраторов: полный CRUD-функционал для пользователей с фильтрацией и сортировкой.

Для всех: ведение категорий и учет платежей.

Визуализация и отчетность: Гибкая аналитика с помощью диаграмм и экспорт отчетов в Excel, Word и PDF с автоматическим подведением итогов.

Интерфейс: Адаптивный UI со сменой тем, подсказками и двухуровневой валидацией данных.

Тестирование: Приложение полностью протестировано, включая логику, безопасность и удобство использования.
